# JumpToOtherEditer 使用说明

## 简介

JumpToOtherEditer 是一款 IntelliJ IDEA 插件，让你可以快速在外部 AI 编辑器中打开当前文件，并自动定位到光标所在行。

## 支持的编辑器

| 编辑器 | 命令 |
|--------|------|
| VS Code | `code` |
| Cursor | `cursor` |
| Antigravity | `antigravity` |
| Qoder | `qoder` |
| Trae | `trae` |
| CodeBuddy | `buddy` |

## 使用方法

1. 在 IDEA 中打开任意文件
2. **右键点击** → **Jump to External Editor**
3. 选择目标编辑器

文件将以项目模式打开，并自动定位到当前行。

## 配置

进入 `Settings` → `Tools` → `JumpToOtherEditer`

- **启用/禁用**：控制编辑器是否显示在菜单中
- **自定义路径**：如果命令无法识别，可手动指定可执行文件路径

## 使用提示

### 💡 Tip 1: 安装时添加到 PATH
安装编辑器时，请勾选 **"添加命令到 PATH"** 选项。

### 💡 Tip 2: 重启 IDEA
如果先开 IDEA 后装编辑器，需要 **重启 IDEA** 以获取最新环境变量。

### 💡 Tip 3: 手动配置路径
如果命令无法识别，可在设置中配置可执行文件完整路径。

### 💡 Tip 4: Trae 特别说明
Trae 首次打开时，需点击 **"安装 trae 命令"** 按钮。

## 常见问题

**Q: 点击后没反应？**
- 检查编辑器是否已安装
- 检查是否已添加到 PATH
- 尝试重启 IDEA

**Q: 提示"不是内部或外部命令"？**
- 在设置中手动配置可执行文件路径
- 或重新安装编辑器，勾选"添加到 PATH"

## 常见安装路径 (Windows)

```
VS Code:    %LOCALAPPDATA%\Programs\Microsoft VS Code\Code.exe
Cursor:     %LOCALAPPDATA%\Programs\cursor\Cursor.exe
Qoder:      %LOCALAPPDATA%\Programs\Qoder\Qoder.exe
Trae:       %LOCALAPPDATA%\Programs\Trae\Trae.exe
CodeBuddy:  %LOCALAPPDATA%\Programs\CodeBuddy\CodeBuddy.exe
```
